# math-lang
Basic Math Language

## Running
just run `Main` found in `org`
## Features
* Basic Math Functions
  * (+ 1 2)
  * (* 3 4)
  * (exp 3)
  * (ln 1)
* Defining Symbols
  * (def A 123)
  * (+ A 1)
  * e and tau are already defined symbols
* Defining Functions (wip)
  * Doesn't work with defined symbols
  * (fun myFun [a b] (+ a b))
  * (myFun 1 2)
